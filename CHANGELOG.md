# Changelog

All notable changes to Cortex Memory will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of Cortex Memory System
- `cortex init` command to initialize projects
- `cortex add-memory` command for documenting learnings
- `cortex add-rule` command for creating project rules
- `cortex list` command to view memories and rules
- `cortex sync` command for memory consolidation
- Template system for structured memories
- CLAUDE.md integration for Claude Code
- Automatic folder structure creation
- Interactive CLI prompts
- Colored terminal output
- Example memories and rules
- Comprehensive documentation
